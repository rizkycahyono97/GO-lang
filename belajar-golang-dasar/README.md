## Cara Menggunakan

1.  **Instal Go** jika belum terpasang di sistem.
2.  Clone atau download repository ini.
3.  Jalankan setiap file dengan perintah:
    
    ```sh
    go run *nama_file.go*
    ```
    
4.  Untuk mengelola modul, gunakan perintah:
    
    ```sh
    go mod tidy
    ```
